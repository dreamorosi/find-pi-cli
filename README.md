# find-pi-cli

> Returns the IP address of a Raspberry Pi on a local network.

Requires arp (address resolution protocol) to work.

## Install
```
$ npm install --global find-pi-cli

or

$ yarn global add find-pi-cli
```

## Usage
```
$ find-pi --help

  Usage
    $ find-pi

  Example
    $ find-pi
    192.168.1.2
```
